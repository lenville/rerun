Generates code in re_query so that cached results can easily be converted to ready-to-use archetypes.

That code needs to be generated directly in the caching crates as it needs access to the cached queries and results as well as the promise resolving machinery. Generating such code in the usual places would result in one giant cycle dependency chain.
